This summer I've been doing a major overhaul of this website to change it from a very simple, static HTML, centered layout site into an awesome, new, dynamic and responsive site built on a WordPress (the most common CMS on the web). I have be working carefully along side the "client" (a longtime friend), Aaron Isaacson, who is the owner of the studio.

I am pleased with the layout, since this was my first time trying a single page site (somewhat inspired by other trendy parallax sites out there). This is also some of my best design work yet.

Time

Check it out

During a Scripting Languages class, I was given the task of creating a form that would take user input and output a visualization of the data. This was also my first exposure to jQuery Mobile, which was quick fun to experiment with.

The idea behind "Time" is to show people examples of how long events in life actually are through the use of colorful, graphical representations.
